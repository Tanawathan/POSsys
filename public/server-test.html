🚀 手動啟動伺服器步驟：

  1. 開啟命令提示字元（CMD）或PowerShell
  2. 切換到專案目錄：cd "c:\Users\fiend\OneDrive\桌面2\tanawat-order-api-main"
  3. 啟動伺服器："C:\Program Files\nodejs\node.exe" proxy-server.js
  4. 看到 "🚀 伺服器已啟動於 http://localhost:3000" 表示成功
  5. 然後重新整理此頁面測試連線
